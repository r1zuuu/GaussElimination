# GaussElimination

## Overview

GaussElimination is a Python project that solves systems of linear equations using the Gaussian elimination method.

The application generates equations, processes them numerically, and presents the results in a clear and structured format.

## Features

- generation of equation systems
- solving systems with Gaussian elimination
- command-line interaction
- formatted console output
- export of results to a text file
- modular Python code structure

## Tech Stack

- Python
- NumPy
- Rich
